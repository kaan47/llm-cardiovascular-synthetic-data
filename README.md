# llm-cardiovascular-synthetic-data
Repository containing LLM-generated synthetic cardiovascular datasets and a comprehensive analytical framework for assessing their structural coherence, statistical realism, and machine learning utility using a real-world reference cohort and an integrative ensemble meta-learning approach for cardiovascular risk prediction.

## Datasets

### Real-World Reference Data
The ground truth for this study is the **Heart Disease Dataset (Comprehensive)**, which provides the statistical foundation for the synthetic generation process.
* **Source:** [IEEE Dataport - Heart Disease Dataset](https://ieee-dataport.org/open-access/heart-disease-dataset-comprehensive)

### Synthetic Datasets
* `chatGPT_cardio.csv`: Synthetic cardiovascular records generated by GPT.
* `gemini_cardio.csv`: Synthetic cardiovascular records generated by Gemini.

## Methodology and Framework

The predictive modeling component utilizes advanced ensemble and meta-learning architectures. The ensemble machine learning framework implemented in this study is based on the methodology described in:

* **Reference:** [Integrative Ensemble and Meta-Learning Frameworks for High-Precision Cardiovascular Risk Prediction](https://link.springer.com/article/10.1007/s13721-026-00742-2)

## Repository Structure

| File | Description |
| :--- | :--- |
| `analysis.ipynb` | Comprehensive analytical evaluation of the synthetic and real datasets, including statistical comparisons, distributional assessments, feature relationship analysis, and visualization of dataset characteristics. |
| `Gpt_CardioMetaEnsemble.ipynb` | Implementation of the integrative meta-ensemble machine learning framework using the GPT-generated synthetic dataset, including model training and hyperparameter optimization. |
| `Gemini_CardioMetaEnsemble.ipynb` | Implementation of the integrative meta-ensemble machine learning framework using the Gemini-generated synthetic dataset, including model training and hyperparameter optimization. |
| `train_and_validate_chatGPT_cardio.ipynb` | Model development using the GPT synthetic dataset for training, followed by validation against the real-world cardiovascular reference dataset. |
| `train_and_validate_gemini_cardio.ipynb` | Model development using the Gemini synthetic dataset for training, followed by validation against the real-world cardiovascular reference dataset. |

## Usage

This repository provides the synthetic datasets and analysis notebooks used in our study.  
All analyses can be reproduced by running the Jupyter notebooks included in the repository.

Typical workflow:

1. Explore the datasets and statistical analyses:
   - `analysis.ipynb`

2. Train the meta-ensemble models on synthetic datasets:
   - `Gpt_CardioMetaEnsemble.ipynb`
   - `Gemini_CardioMetaEnsemble.ipynb`

3. Evaluate synthetic-to-real generalization:
   - `train_and_validate_chatGPT_cardio.ipynb`
   - `train_and_validate_gemini_cardio.ipynb`

The notebooks contain the full pipeline for data analysis, visualization, model training, hyperparameter optimization, and validation experiments.
